# bio-nio-study

bio:
    fileTransfer : 通过bio完成文件传输
    moreClient   : 通过bio完成多客户端与服务端通信
    oneMessage   : 通过bio完成客户端发送一条消息给服务端

channel: nio的Channel功能演示
nio:  通过nio完成客户端、服务端通信、Buffer功能测试
nio_chat:   通过nio技术完成群聊功能。ChatServer为服务端，端口为9999；ChatClient为客户端
